# Youtube Downloader
>Só mais um downloader de mp3 do youtube

[![NPM Version][npm-image]][npm-url]

![](../header.png)

## Instalação

```sh
npm install && npm link
```
## Uso
Após a instalação,digite via linha de comando:
```sh
baixar url {url}
```
Depois procure pelo arquivo mp3 no seu diretório
